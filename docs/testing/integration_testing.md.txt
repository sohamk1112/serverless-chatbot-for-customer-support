# Integration Testing

## Step 1: Test Lex and Lambda Integration
1. Use the Lex console to test each intent.
2. Verify that the Lambda function is called and returns the correct response.

## Step 2: Test API Gateway and Lambda Integration
1. Use a tool like Postman to send requests to the API Gateway endpoint.
2. Verify that the Lambda function is invoked and returns the expected response.

## Step 3: Test End-to-End Workflow
1. Open the front-end application.
2. Test the complete workflow from sending a message in the chat interface to receiving a response from the bot.
3. Ensure all components work together seamlessly.
